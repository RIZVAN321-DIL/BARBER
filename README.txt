Бот для парикмахера (финальная исправленная версия)

Функции:
- Клиент: запись через календарь или "Ближайшее время", имя и телефон опционально (кнопка Пропустить).
- Клиент может отменить запись за 2+ часа до начала.
- Админ: просмотр всех записей, быстрая запись по звонку (без ввода данных), отмена по ID с подтверждением, статистика.
- Напоминания за 24 часа и за 2 часа.
- Гибкий график (задаётся в config.py).

Переменные окружения (задать на хостинге):
BOT_TOKEN, ADMIN_ID, MASTER_PHONE

Запуск: python main.py
