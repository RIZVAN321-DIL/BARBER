Бот для парикмахера-одиночки (финальная версия)

Функции:
- Клиент: запись за 10 секунд, мои записи, отмена (за 2+ часа)
- Админ: все записи, быстрая запись по звонку (без ввода имени/телефона), отмена по ID, статистика
- Автоматические напоминания за 24 часа и за 2 часа
- Защита: 1 активная запись, блокировка поздней отмены

Переменные окружения (настройте на хостинге):
BOT_TOKEN=ваш_токен
ADMIN_ID=ваш_цифровой_id
MASTER_PHONE=номер_телефона_мастера

Запуск: python main.py
Для хостинга: загрузите ZIP, укажите точку входа main.py.
