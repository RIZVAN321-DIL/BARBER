✂️ Бот для парикмахера — финальная версия

✅ Функции:
– Клиент: запись (календарь / ближайшее время), имя/телефон опционально
– Отмена записи клиентом за 2+ часа
– Админ: все записи (с телефонами), быстрая запись по звонку, отмена по ID, статистика, чёрный список (/ban, /unban)
– Напоминания за 24ч и 2ч
– График работы 8:00–17:00 (слот 1 час) — меняется в config.py

⚠️ Перед запуском:
1. Установить зависимости: pip install -r requirements.txt
2. Ввести переменные окружения (на хостинге): BOT_TOKEN, ADMIN_ID, MASTER_PHONE, TZ=Europe/Moscow
3. Запуск: python main.py

Все данные хранятся в barber.db.
