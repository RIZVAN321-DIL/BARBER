Бот для парикмахера-одиночки с защитой от спама.

Функции:
- Стрижка, борода, комплекс
- Календарь на 60 дней
- Слоты по 1 часу (08:00-17:00)
- Клиент может отменить запись (только если до начала >2 часов)
- Админ отменяет любую запись
- Напоминание за 1 час
- Защита: лимит 1 активная запись, чёрный список, rate limit (3 попытки/мин), проверка бана
- Команды /ban и /unban для админа

Запуск:
1. Установите зависимости: pip install -r requirements.txt
2. В config.py укажите BOT_TOKEN, ADMIN_ID, MASTER_PHONE
3. Запустите: python main.py

Для хостинга: заархивируйте все файлы в ZIP (без папки) и загрузите.
